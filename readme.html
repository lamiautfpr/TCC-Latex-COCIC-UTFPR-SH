[Federal University of Technology — Paraná] UTFPR-Thesis

Go to PDF file of this LaTeX template

This LaTeX template (unofficial) allows the production of academic works (doctoral, master, and bachelor theses, among others) of the Federal University of Technology — Paraná (UTFPR). It was developed using the memoir LaTeX class, to meet the requirements of the Brazilian Association of Technical Standards (ABNT) for the development of Brazilian technical and scientific documents. Also, several code snippets of the abnTeX2 LaTeX class and others developed by users of the TeX-LaTeX Stack Exchange were used.

Este modelo LaTeX (não oficial) permite a produção de trabalhos acadêmicos (teses, dissertações, trabalhos de conclusão, entre outros) da Universidade Tecnológica Federal do Paraná (UTFPR). Foi desenvolvido utilizando a classe LaTeX memoir, de modo a atender aos requisitos da Associação Brasileira de Normas Técnicas (ABNT) para o desenvolvimento de documentos técnicos e científicos brasileiros. Além disso, diversos trechos de códigos da classe LaTeX abnTeX2 e outros desenvolvidos por usuários do TeX-LaTeX Stack Exchange foram usados.

1.1. Compilação dos arquivos

No diretório do projeto, abra um prompt de comando (no Windows) ou um console (no Linux) e digite make help para obter informações a respeito dos comandos de compilação dos arquivos do modelo LaTeX:

  • ./make.bat (Batchfile para Compilação no Prompt de Comando do Windows).
  • ./makefile (Makefile para Compilação no Console do Linux).

5. Dicas de uso

Sempre que for manipular e/ou substituir um dos arquivos constituintes do projeto, mantenha uma cópia do original num local seguro e/ou renomeie a cópia do original para poder ser seguida como um exemplo no desenvolvimento do seu próprio arquivo.

O único formato de imagens que pode ser inserido ao compilar com LaTeX é o Encapsulated PostScript (EPS). Se for compilar com PDFLaTeX, para produzir um PDF diretamente, tem-se uma escolha mais ampla de formatos de imagens para inserir:

  • JPG, amplamente utilizado na Internet, em câmeras digitais, etc., sendo a melhor escolha para fotos.
  • PNG, um formato sem perdas e muito comum (ainda que não tanto como JPG), sendo a melhor escolha para diagramas (se não for possível gerar uma versão vetorial) e capturas de tela.
  • PDF, é amplamente utilizado para documentos, mas pode ser usado para armazenar imagens também; suporta imagens vetoriais e mapa de bits, mas não é recomendado para o último, pois JPG ou PNG irá fornecer o mesmo resultado com menor espaço em disco.
  • EPS, pode ser usado com ajuda do pacote epstopdf, que só precisa estar instalado; portanto, não há necessidade de carregá-lo no documento.
